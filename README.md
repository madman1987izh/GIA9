О проекте
Система анализа результатов ОГЭ/ЕГЭ — это desktop-приложение для автоматизации обработки и анализа результатов государственных экзаменов. Программа позволяет импортировать данные из PDF-протоколов, сохранять их в базу данных, формировать статистику и экспортировать отчёты в PDF.

Основные возможности
Импорт данных из PDF-протоколов ОГЭ/ЕГЭ (автоматическое или ручное определение предмета)

Защита от дублирования — проверка хеша файла перед импортом

Управление экзаменами — просмотр, удаление, поиск дубликатов

Ведение пересдач — добавление и сохранение результатов пересдач

Статистика — распределение оценок, успеваемость, качество знаний (по предметам и классам)

Экспорт отчётов в PDF:

Отчёт по ученикам (таблица с группировкой по классам)

Детальный отчёт по классам (с полной информацией о каждом ученике)

Статистический отчёт (сводная информация)

🛠 Технологии
Python 3.10+

PyQt6 — графический интерфейс

SQLite — база данных

pdfplumber — извлечение данных из PDF

reportlab — генерация PDF-отчётов

pandas — экспорт в Excel

🔧 База данных
При первом запуске автоматически создаётся файл exam_results.db со следующими таблицами:

exams
id — ID экзамена

subject — предмет

exam_date — дата экзамена

region — регион

file_hash — хеш файла (для защиты от дубликатов)

exam_attempts
id — ID попытки

exam_id — ссылка на экзамен

last_name, first_name, middle_name — ФИО

class — класс

primary_score — первичный балл

grade — оценка

subject — предмет

exam_date — дата экзамена

student_unique_id — уникальный идентификатор ученика
